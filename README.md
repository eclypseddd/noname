# MyPet telegram Bot

Телеграм бот где ты можешь сгладить своё одиночество, проводя время с питомцем.

## Сведения
- бот написан на Python
- для создания проекта использовалась библиотека telebot
- над проектом работал один человек

## Комманды

#### /start

#### /create
#### /sleep
#### /feed
#### /play
#### /status
